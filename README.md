## Install dependencies

`cd <project_name>`

`yarn install`

## Start your mongoDB local server

`mongod`

## Build and run the project

`yarn run build`

`yarn start`

Navigate to `_http://localhost:3000_`

## Run tests

`npm run build`

`npm run test`
